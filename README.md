# Bank Management System (Java + JDBC)

## Overview
This project is a simple console-based Bank Management System developed in Java using JDBC and MySQL. It allows users to add customers, create bank accounts, and deposit money while storing all records in a MySQL database.

---

## Features
- Add new customers
- Create bank accounts
- Deposit money into accounts
- Store transaction records
- MySQL database connectivity using JDBC

---

## Technologies Used
- Java
- JDBC
- MySQL
- Scanner Class
- PreparedStatement

---

## Database Tables
The project uses the following tables:

1. **Customers**
2. **Accounts**
3. **Transactions**
